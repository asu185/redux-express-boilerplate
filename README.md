# Redux / Express boilerplate

Required command line tools
```javascript
$node -v
6.11.0

$npm -v
3.10.10
```

Required global npm packages
```javascript
$npm install -g webpack
$npm install -g pm2
```

```
$npm install
```

For dev mode
```
$npm run dev
```

For prod mode
```
$npm run build
$npm start
```
